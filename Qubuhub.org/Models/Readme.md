# QUBUHUB Governance Pack

This repository contains the public governance and operational trust documents for QUBUHUB AI systems.

## Included Documents
- `model-behavior.md`
- `agent-terms.md`
- `agent-safety.md`
- `security-policy.md`

## Purpose
These documents define how QUBUHUB systems are expected to behave, what use is permitted, what safety boundaries apply, and how security issues are handled.

## Governing Principle
If a system is powerful but not governable, it is not production-ready.
